PREBUILT IMAGES

Download node
    docker pull node
    or
    docker run node

Display all processes
    docker ps -a

Display running processes
    docker ps

Display last process running or not
    docker ps -l

Expose an interactive session
    docker run -it node
    CTRL+C x 2

Help
    docker --help
    docker ps --help
    docker logs --help
